# 查婷作品集 - GitHub Pages 部署包

使用方法：

1. 新建 GitHub 仓库，例如 `chating-portfolio`。
2. 解压本 ZIP。
3. 把解压后的全部文件上传到仓库根目录：
   - index.html
   - .nojekyll
   - README.md
4. 打开仓库 Settings → Pages。
5. Source 选择 `Deploy from a branch`。
6. Branch 选择 `main`，文件夹选择 `/root`，保存。
7. 等待 GitHub Pages 生成访问链接。

更新网站时，只需要替换仓库根目录里的 `index.html`。
